# ASCII Art Editor (vscode-extension-ascii-sketch)

Visual Studio Code エディタ上でアスキーアートを快適に作成・編集するための拡張機能です。

## 主な機能

VS Code 標準エディタの強力な機能を活かしつつ、`.ascii` ファイルを扱うための専用のサポートを提供します。

* **専用の言語サポート機能**: `.ascii` ファイルを開いた際、自動的に等幅フォントを適用し、上書きモード時にはカーソルの形状をブロックカーソルに変更することで、アスキーアートの描画に最適な環境を設定します。
* **キャンバス生成 (`ASCII Art: Create Canvas`)**: 80文字×20行の半角スペースで埋められた「未保存の新規ドキュメント (Untitled)」を即座に生成し、すぐに絵を描き始められるまっさらなキャンバスを用意します。
* **スマート枠線生成 (`ASCII Art: Generate Box`)**: 矩形選択（`Shift+Alt+Drag` またはマルチカーソル）で範囲を指定してこのコマンドを実行すると、選択範囲の境界線をアスキー文字（`+`, `-`, `|`）に置き換え、瞬時に枠線を描画します。
* **枠線の自動修整・アラインメント (`ASCII Art: Align Box`)**: 枠内に全角文字（日本語や絵文字など）を入力して縦の罫線がズレてしまった場合、このコマンドを実行することで「East Asian Width（東アジアの文字幅）」を再計算し、右側の枠線（`│`）の位置を自動で綺麗に揃え直します。

## 動作要件

特別な要件や外部依存はありません。VS Code 本体に組み込まれたエディタ機能を用いて動作します。

## 拡張機能の設定について

この拡張機能は、`.ascii` ファイルを編集する際に以下のデフォルト設定を自動適用します：

```json
"[ascii-art]": {
    "editor.cursorStyle": "line",
    "editor.overtypeCursorStyle": "block"
}
```

## 既知の制限事項・仕様

* **上書きモードの制限**: 線画を描く際は標準機能の上書きモード（`Insert` キーでトグル）の利用を推奨していますが、VS CodeのAPIの制限により、**ファイルを開いた瞬間に強制的に上書きモードにする機能は提供していません**。描画を始める際は手動で `Insert` キーを押し、カーソルがブロック型（`█`）になったことを確認してください。
* **IME（日本語入力）との兼ね合い**: 日本語入力などの変換処理を壊さないよう、文字入力イベント（`type` コマンド）のリアルタイムフックは行わず、全ての入力処理を VS Code 側に委ねています。そのため、「全角文字を打った瞬間に枠線が自動で押し出される」ようなリアルタイム修整機能は搭載していません。入力後に `Align Box` コマンドを実行してレイアウトを整えてください。
